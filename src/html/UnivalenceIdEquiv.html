The identity equivalence creates the trivial path.

::

     ua-id
       : ∀ {ℓ : Level} {A : Type ℓ}
       --------------------
       → ua idEqv ≡ refl A

     ua-id {A = A} =
       begin
         ua idEqv
           ≡⟨ ap ua (sameEqv (refl id)) ⟩
         ua (idtoeqv (refl A))
           ≡⟨ ua-η (refl A) ⟩
         refl A
       ∎
